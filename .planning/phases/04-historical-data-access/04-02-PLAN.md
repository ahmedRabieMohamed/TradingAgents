- get_corporate_actions(market_id, symbol, start_date, end_date) using EodhdClient.get_dividends and get_splits.

- Normalize dividends and splits into DividendAction and SplitAction with Optional fields; tolerate missing optional fields.
- Build freshness using the oldest fetched_at and the max age_seconds across both datasets; cache_status "fresh" if both are fresh else "miss".
- Raise ApiError(status 503, code "CORP_ACTIONS_UNAVAILABLE") if both provider calls fail and no cached data exists.
